# 税务师备考中心 🏛️

> 基于2025年税务师考试教材整理的知识可视化与智能刷题系统

**[🚀 在线体验](https://brucechang815-dev.github.io/tax-advisor-study/)**

---

## 📚 项目介绍

本项目是一套面向**2025年税务师考试**的全科备考工具，包含：

1. **交互式思维导图** — 按科目梳理知识脉络，支持展开/收缩、搜索定位
2. **智能题库系统** — 四种题型覆盖，错题自动标记、反复强化
3. **学习进度追踪** — 可视化展示各科目备考状态

---

## 📖 五大科目

| 科目 | 核心内容 | 颜色标记 |
|------|---------|--------|
| **税法（一）** | 增值税、消费税、关税、资源税、土地增值税 | 🔴 红色 |
| **税法（二）** | 企业所得税、个人所得税、房产税、印花税、契税 | 🟠 橙色 |
| **财务与会计** | 财务管理、投资决策、会计核算、财务报表 | 🟢 绿色 |
| **涉税服务实务** | 税务登记、纳税申报、税务代理、纳税筹划 | 🔵 蓝色 |
| **涉税服务相关法律** | 民法、合同法、公司法、行政法、税收征管法 | 🟣 紫色 |

## ✨ 功能特色

### 🗺️ 思维导图
- 按科目→章→节分层展示，最深支持4-5级节点
- 点击节点展开详情面板，查看知识点说明
- 支持全展开/分级收缩/复位视图
- 内置搜索：输入关键词高亮定位知识点
- 鼠标拖拽/滚轮缩放
- 一键跳转至相关题目

### ✏️ 题库系统
- **四种题型**：单选题、多选题、判断题、计算/简答题
- **多种练习模式**：章节练习、随机练习、错题复习、模拟测试
- **详细解析**：每道题都有答案解析和解题思路
- **错题强化**：做错后自动加入错题本，后续练习反复出现，连续答对3次才移出
- 支持按科目、章节、题型筛选

### 📊 进度追踪
- 各科目答题数量、正确率、练习覆盖率
- 错题本：按错误次数排序，可一键练习所有错题
- 数据本地存储，无需注册登录

## 🚀 快速开始

### 本地运行

```bash
# 克隆项目
git clone https://github.com/brucechang815-dev/tax-advisor-study.git

# 进入目录
cd tax-advisor-study

# 直接用浏览器打开（推荐用本地服务器避免跨域问题）
# 方式一：Python
python -m http.server 8080

# 方式二：Node.js
npx serve .

# 然后访问 http://localhost:8080
```

### 部署到 GitHub Pages

1. Fork 本项目，或创建新仓库并上传文件
2. 进入仓库 → `Settings` → `Pages`
3. Source 选择 `main` 分支，目录选择 `/ (root)`
4. 点击 Save，几分钟后即可访问 `https://brucechang815-dev.github.io/tax-advisor-study/`
